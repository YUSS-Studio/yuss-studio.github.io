List Arts Center

Cultural Visual Identity
List Arts Center is the contemporary art museum on campus at MIT. This project looks at the semantic value of the name/word “List” to create new organizational methodologies that reflect the Center's commitment to the circulation of knowledge.

Tools
InDesign, Illustrator, Photoshop, P5.JS, Processing
